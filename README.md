# ProjectOps Dashboard

Dashboard de operaciones para gestión de proyectos construido con Angular.

## Descripción

Aplicación de gestión y monitoreo de operaciones de proyectos con módulos de:
- **Projects** - Tablero Kanban y gestión de proyectos
- **Tasks** - Lista y gestión de tareas
- **Team** - Vista general del equipo
- **Metrics** - Dashboard de métricas

## Instalación

```bash
pnpm install
```

## Uso

```bash
pnpm start
```
